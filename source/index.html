关于

网站的基本样式参考了Dandy的主页，自己陆续设计过几个，发觉都没有这个来得满意。

趁着 Bootstrap 4 发布之际，更新了自己的版本，并且加入了扁平化的设计。

该网页采用栅格化响应式布局，应用了多种实验性 CSS 元素和大量动画效果，

并支持 Retina 高清屏幕和 iOS Web App，

还有基于 HTML 5 的离线缓存和 Canvas 绘图等技术。
